# Wanderlust 🌍

A full-stack travel listing and booking platform built with modern web technologies.  
Wanderlust allows users to explore destinations, view listings, and interact with travel services through a clean and responsive interface.

---

## Features

- Browse travel listings
- Search and filter destinations
- User authentication (signup/login)
- Add and manage reviews
- Location-based listings (Map integration)
- Image uploads (Cloud storage)
- Flash messages & error handling

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** Passport.js
- **Storage:** Cloudinary
- **Maps:** OpenCage API
